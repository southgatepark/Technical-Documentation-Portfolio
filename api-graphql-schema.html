Query Types

Query product

Retrieve a single product by its unique identifier. Returns detailed product information including pricing, inventory, and associated metadata.

id *
ID!
Unique product identifier
Query Example graphql 
query {
  product(id: "prod_123") {
    id
    name
    description
    price {
      amount
      currency
    }
    inventory {
      available
      reserved
    }
    images {
      url
      alt
    }
  }
}
Response Example json 
{
  "data": {
    "product": {
      "id": "prod_123",
      "name": "Wireless Headphones",
      "description": "Premium noise-cancelling headphones",
      "price": {
        "amount": 29999,
        "currency": "USD"
      },
      "inventory": {
        "available": 45,
        "reserved": 3
      }
    }
  }
}
Query products

Search and filter products with pagination support. Supports filtering by category, price range, availability, and full-text search.

filter
ProductFilter
Filter criteria for products
sort
SortOrder
Sort order (PRICE_ASC, PRICE_DESC, NAME, NEWEST)
limit
Int
Number of results per page (default: 20, max: 100)
offset
Int
Number of results to skip for pagination
Query Example graphql 
query {
  products(
    filter: { 
      category: "electronics",
      priceRange: { min: 10000, max: 50000 },
      inStock: true
    },
    sort: PRICE_ASC,
    limit: 10
  ) {
    totalCount
    edges {
      node {
        id
        name
        price {
          amount
          currency
        }
      }
    }
    pageInfo {
      hasNextPage
      endCursor
    }
  }
}
M

Mutations

Mutation createOrder

Create a new order with line items. This mutation validates inventory availability, calculates totals including tax and shipping, and returns the created order object.

input *
CreateOrderInput!
Order creation data including items and shipping info
Mutation Example graphql 
mutation {
  createOrder(input: {
    customerId: "cust_456",
    items: [
      { productId: "prod_123", quantity: 2 },
      { productId: "prod_789", quantity: 1 }
    ],
    shippingAddress: {
      street: "123 Main St",
      city: "San Francisco",
      state: "CA",
      zipCode: "94102",
      country: "US"
    }
  }) {
    order {
      id
      orderNumber
      status
      total {
        amount
        currency
      }
      createdAt
    }
    errors {
      field
      message
    }
  }
}
Mutation updateProduct

Update an existing product's information. All fields except ID are optional. Only provided fields will be updated.

id *
ID!
Product identifier to update
input *
UpdateProductInput!
Fields to update on the product
Mutation Example graphql 
mutation {
  updateProduct(
    id: "prod_123",
    input: {
      name: "Premium Wireless Headphones",
      price: { amount: 27999, currency: "USD" }
    }
  ) {
    product {
      id
      name
      price {
        amount
        currency
      }
      updatedAt
    }
  }
}
